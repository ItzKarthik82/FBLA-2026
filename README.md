## Getting Started

1. Open `FBLA 2026/pages/index.html` in your browser
2. Or run a local server:
   ```bash
   cd "FBLA 2026/pages"
   python3 -m http.server 8000
   # Visit http://localhost:8000/index.html
   ```

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript (ES6+)
- localStorage for data storage
- No backend, no dependencies - runs entirely client-side

## Data Storage

All user data is stored locally in the browser using localStorage (accounts, sessions, progress, etc.).
